# Tatumdale Website - Design Specification

## Brand Assets
- **Primary Color:** #F02878 (Hot Pink/Coral)
- **Secondary Color:** #01285C (Navy Blue)
- **Accent Colors:** Black, White, Grays
- **Logos:** Dark logo (navy), Light logo (white on transparent)
- **Favicon:** Navy crab with pink accent

## Site Structure (from Nova's Google Doc)
1. **Hero** - Tagline, intro, primary CTA
2. **About/Framework** - Discover Deeply, Design Deliberately, Deliver Dynamically
3. **Services** - What Tatum helps with
4. **Experience** - Companies worked for + certifications
5. **Products** - Hoursly and PMOS apps
6. **Day Rate Calculator** - Embedded iframe from Vercel
7. **Contact** - Email + Calendar booking CTA

## Three Design Versions

### Version 1: "Bold Editorial"
- **Vibe:** Confident, magazine-style, attention-grabbing
- **Colors:** Navy dominant (#01285C), white sections, pink accents (#F02878)
- **Typography:** Large bold headlines, clean sans-serif
- **Layout:** Asymmetric, alternating section backgrounds, bold typography
- **Motion:** Scroll-triggered reveals, animated underlines, sticky nav

### Version 2: "Clean Minimal"
- **Vibe:** Refined, professional, breathable
- **Colors:** Predominantly white, navy text, strategic pink accents
- **Typography:** Medium weight, generous letter-spacing, high line-height
- **Layout:** Center-aligned, generous whitespace, 3-column grids
- **Motion:** Subtle fade-ins, smooth scroll, hover underlines

### Version 3: "Dark Sophisticated"
- **Vibe:** Premium, tech-forward, modern
- **Colors:** Deep navy backgrounds, white text, pink gradient accents
- **Typography:** Bold white headlines, slight letter-spacing
- **Layout:** Full-bleed sections, glass-morphism cards, gradient accents
- **Motion:** Gradient animations, glow effects, parallax-style backgrounds

## Technical Stack
- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS with custom theme
- **Animation:** Framer Motion
- **Fonts:** Inter (Google Fonts)
- **Deployment:** Vercel (assumed from day-rate-calculator pattern)

## Key Sections
- Sticky navigation with logo + links + CTA button
- Hero with centered content + primary CTA
- Framework (3 pillars) cards
- Services grid
- Experience logo strip
- Products showcase
- Day Rate Calculator iframe
- Contact section with dual CTAs
- Minimal footer
